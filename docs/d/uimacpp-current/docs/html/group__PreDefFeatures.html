Predefined Features

Detailed Description

For each predefined feature the system provides a string constant for the names of the feature.

Programmers are strongly encouraged to use them instead of string literals. E.g. write CAS::FEATURENAME_LEMMAENTRIES instead of "UIMA_Feature_LemmaEntries"
