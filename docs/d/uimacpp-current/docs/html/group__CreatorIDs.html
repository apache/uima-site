Creator IDs

Detailed Description

Creator IDs are used with uima::Type::getCreatorID() and uima::Feature::getCreatorID() The ID shows which component has created a type or feature.

They are intended for information/debug/display purposes. Currently two creator IDs are predefined:

  • CAS::ustrCREATOR_ID_CAS for the core types/features created by the CAS (e.g. Integer)
  • CAS::ustrCREATOR_ID_CAS for the basic linguistic types/features created by the CAS (e.g. Annotation)

For types/features not created by CAS the creator id is the name of the component/annotator
